Formulario de registro con Validaciones HTML

Formulario web orientado a la recopilación estructurada de datos, implementando validaciones nativas de HTML para garantizar la integridad de la información sin depender de JavaScript.

A nivel de estilos, se trabajó en la consistencia visual de los campos, estados de interacción y organización del layout, priorizando claridad, accesibilidad y una experiencia de usuario fluida.

Lo que aprendí:

  • Implementación de validaciones nativas como required, pattern, min y max, aprovechando el comportamiento del navegador.
  • Uso de fieldset y legend para estructurar formularios de forma semántica y accesible.
  • Estilización de formularios manteniendo consistencia entre inputs, selects y botones, incluyendo estados como :focus y :valid/:invalid.
  • HTML5
  • CSS3

Contenido Multimedia y Responsive

Implementación de contenido multimedia optimizado para la web, incluyendo imágenes responsivas, uso del elemento picture para art direction y carga eficiente de recursos como video y audio.

Desde el enfoque de CSS, se trabajó en la adaptación visual del contenido a distintos dispositivos, asegurando layouts flexibles y una correcta integración de elementos multimedia dentro del diseño.

Lo que aprendí:

  • Uso de srcset y sizes para servir imágenes adaptadas según el dispositivo.
  • Implementación del elemento picture para control de versiones de imagen (art direction).
  • Aplicación de técnicas responsive con CSS para adaptar imágenes y contenedores multimedia sin romper el layout.
  • HTML5
  • CSS3
